Check contents for the ADT normalization step. Contents are stored inside a adt_normalization HDF5 group at the root of the file. The adt_normalization group itself contains the parameters and results subgroups.

No ADT data was available prior to version 2.0 of the format, so the adt_normalization group may be absent in pre-v2.0 files.

No contents are mandated for parameters.

If adt_in_use = false, results should be empty.

If adt_in_use = true, results should contain:

  • size_factors, a float dataset of length equal to the number of cells, containing the size factor for each cell.
Parameters
handleAn open HDF5 file handle.
num_cellsNumber of cells in the dataset.
adt_in_useWhether ADTs are being used in this dataset.

Returns
If the format is invalid, an error is raised.
