std::string kanaval::choose_clustering::validate ( const H5::H5File &  handle)
inline

Check contents for the clustering choice. Contents are stored inside a choose_clustering HDF5 group at the root of the file. The choose_clustering group itself contains the parameters and results subgroups.

parameters should contain:

  • method: a scalar string specifying the clustering method to use. This is currently either snn_graph or kmeans.

results is empty.

Depending on the method, snn_graph_cluster or kmeans_cluster must have non-empty results. Both may also be non-empty, in which case the appropriate clustering is chosen based on method.

Parameters
handleAn open HDF5 file handle.
Returns
The clustering method of choice. If the format is invalid, an error is raised.
